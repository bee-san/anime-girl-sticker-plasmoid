# Anime Girl Sticker Plasmoid

Transparent Plasma 6 desktop widget that displays one user-selected PNG or WebP image without a widget frame, so a character can appear to stand directly on the desktop.

## Install

From this repository root:

```bash
kpackagetool6 -t Plasma/Applet -i ./dev.bee.animegirlsticker
```

## Update

```bash
kpackagetool6 -t Plasma/Applet -u ./dev.bee.animegirlsticker
```

## Remove

```bash
kpackagetool6 -t Plasma/Applet -r dev.bee.animegirlsticker
```

After installing, add `Anime Girl Sticker` from the Plasma widget list, open its settings, choose a transparent PNG or WebP image, and adjust the height.
